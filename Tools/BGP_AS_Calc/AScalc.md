#AScalc

**Objective**
calculate both types AS number, AS plain and AS dotted, fron an ASN value. If the ASN has a dot or colon in it assume as-asdot otherwise as-asplain, return both types asn-asdot followed by asn-asplain. 

**Dependencies**

The script has dependencies on the following python libraries:

 - argparse
 - re

**Command Line Options**

- ASN {{AS number text}} BGP AS Number ASdot or ASplain
